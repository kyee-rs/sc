Ghost File Server
=---------------=

HTTP POST files here:
    curl -F"file=@yourfile.png" {{.host}}
* Plain text in response is the file's URL.

To get a response as JSON:
    curl -F"file=@yourfile.png" --header "Accept: application/json" {{.host}}

Careful! This server is public. Do not upload sensitive files.



Retention Policy
=---------------=
File lifetime is {{.retention}} days. (0 means no limit.)

Blocking Policy
=-------------=
- Tor exit nodes are blocked. ({{.tor}})
- Files larger than {{.maxsize}} megabytes are blocked. (0 means no limit.)
